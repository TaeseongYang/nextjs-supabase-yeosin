# 소다랩(SoDA LAB) MVP PRD

## 🎯 핵심 정보

**목적**: 의료 미용 시술을 카테고리별로 탐색하고, 병원별 시술 상품의 가격·평점·리뷰를 비교한 뒤 상담을 신청할 수 있게 한다. 특히 "의료진/이용서비스/가격/시술효과/시술통증" 5개 속성별로 관리자가 사전에 입력해 둔 리뷰 요약(도넛 차트+bullet)과 해당 속성에 해당하는 개별 리뷰만 필터링해 보여주는 것이 경쟁 서비스 대비 핵심 차별점이다.
**사용자**: 제모, 지방성형, 리프팅 등 미용 시술을 알아보는 일반 소비자로, 모바일 웹에서 여러 병원의 동일 시술 상품을 비교한 뒤 상담을 신청하려는 사람.

## 🚶 사용자 여정

```
1. 카테고리 홈
   ↓ 관심 시술 카테고리 클릭 (12개 중 선택)

2. 시술 상품 리스트
   ↓ 상품 카드 클릭

3. 시술 상품 상세
   ↓ 리뷰 영역 진입

4. 리뷰 종합 뷰 (전체 요약 - 관리자 입력)
   ↓ 5개 속성 태그(의료진/이용서비스/가격/시술효과/시술통증) 중 클릭

   [속성 선택] → 속성별 리뷰 뷰(관리자 입력 요약 + 필터링된 개별 리뷰)
   ↓ "전체 요약 보기" 클릭 시 4번으로 복귀
   ↓

5. "상담받기" 또는 "구매하기" 버튼 클릭
   ↓ 이름/연락처/희망일시 입력

6. 신청 완료 → 시술 상품 상세 페이지로 복귀
```

## ⚡ 기능 명세

### 1. MVP 핵심 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|-------------|------------|
| **[F001]** | 카테고리 홈 | 12개 시술 카테고리(제모/기타/지방성형/가슴성형/치아/리프팅/모발이식/한방/필러/보톡스/안면윤곽·양악/눈성형)를 아이콘 그리드로 노출 | 서비스 진입점이자 핵심 탐색 방식 | 카테고리 홈 페이지 |
| **[F002]** | 시술 상품 리스트 | 선택한 카테고리의 병원별 시술 상품을 썸네일, 병원명·지역, 리뷰 수, 평점, 정가/할인가/할인율과 함께 카드 리스트로 노출 | 여러 병원의 상품 조건을 한눈에 비교하는 핵심 화면 | 시술 상품 리스트 페이지 |
| **[F003]** | 시술 상품 상세 | 옵션, 가격, VAT·마취비·사후관리비 포함 여부 뱃지, 부작용 안내, 상세 이미지를 노출 | 상담/구매 결정에 필요한 핵심 정보 제공 | 시술 상품 상세 페이지 |
| **[F004]** | 리뷰 종합 요약 | 총 리뷰 수, 평균 평점과 함께 관리자가 어드민/DB에 직접 입력한 전체 긍정/부정 비율(도넛 차트)·긍정/부정 bullet 요약, 개별 리뷰 리스트를 노출 | 경쟁 서비스 대비 핵심 차별점(신뢰 가능한 리뷰 요약 제공) | 리뷰 종합 뷰 페이지 |
| **[F005]** | 속성별 리뷰 필터 | "의료진/이용서비스/가격/시술효과/시술통증" 5개 속성 태그 클릭 시, 관리자가 해당 속성 기준으로 직접 입력해 둔 요약(도넛 차트+bullet)과 그 속성에 매핑된 개별 리뷰만 필터링해 노출 | 사용자가 중요시하는 기준으로 좁혀 비교하게 하는 핵심 차별 기능 | 속성별 리뷰 필터 페이지 |
| **[F006]** | 상담 신청 | "상담받기"/"구매하기" 버튼 클릭 후 이름·연락처·희망일시를 입력해 상담을 신청(결제 연동 없이 접수만) | 서비스의 최종 전환 액션 | 상담 신청 페이지 |

### 2. MVP 필수 지원 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|-------------|------------|
| **[F007]** | 개별 리뷰 카드 | 작성자 식별 정보, 평점, 작성일, 신고 버튼, 리뷰 본문을 카드로 노출(신고 처리 자체는 MVP 범위 밖) | 속성별/전체 요약의 근거가 되는 원문을 사용자가 직접 확인하는 수단 | 리뷰 종합 뷰 페이지, 속성별 리뷰 필터 페이지 |
| **[F008]** | 하단 탭 내비게이션 | 홈/병원/이벤트/마이페이지 4개 탭으로 화면을 전환(홈 외 3개 탭은 자리만 존재) | 전체 서비스의 기본 이동 골격 | 카테고리 홈 페이지, 병원 페이지, 이벤트 페이지, 마이페이지 |

### 3. MVP 이후 기능 (제외)

- 리뷰 자동 수집·요약 AI/LLM 파이프라인 — 리뷰 요약(긍정/부정 비율, bullet)은 MVP 기간 동안 관리자가 어드민 화면 또는 DB에서 직접 입력·관리하는 값으로 대체하며, 자동 생성/자동 분석 기능은 도입하지 않는다
- 실제 결제/PG 연동 (상담 신청은 접수까지만)
- 병원 전용 프로필 페이지 및 병원 등록·관리 기능(상품 카드는 병원이 아닌 개별 시술 상품 상세로 직접 연결)
- 이벤트 페이지, 마이페이지 상세 기능(회원 정보, 예약 내역 등)
- 리뷰 신고 처리, 리뷰 작성 기능
- 상품 검색, 고급 정렬·필터

## 📄 페이지별 상세 기능

### 카테고리 홈 페이지

> **구현 기능:** `F001`, `F008` | **메뉴 위치:** 하단 탭 "홈"

| 항목 | 내용 |
|------|------|
| **역할** | 서비스 진입점이자 관심 시술 탐색 시작점 |
| **진입 경로** | 앱 최초 진입, 하단 탭 "홈" 클릭 |
| **사용자 행동** | 12개 카테고리 아이콘 중 관심 있는 시술을 선택 |
| **주요 기능** | • 카테고리 아이콘+라벨 그리드 노출<br>• 카테고리 클릭 시 해당 카테고리로 이동<br>• 하단 탭(홈/병원/이벤트/마이페이지) 노출 |
| **다음 이동** | 카테고리 클릭 → 시술 상품 리스트 페이지 |

| **진입 경로** | 카테고리 홈 페이지에서 카테고리 아이콘 클릭 |

> **구현 기능:** `F003` | **메뉴 위치:** 시술 상품 리스트 페이지에서 진입

| 항목 | 내용 |
|------|------|
| **역할** | 특정 병원의 특정 시술 상품에 대한 상세 정보 제공 |
| **진입 경로** | 시술 상품 리스트 페이지에서 상품 카드 클릭 |
| **사용자 행동** | 옵션과 가격을 확인하고, 포함 사항·부작용 안내·상세 이미지를 훑어본 뒤 리뷰를 확인하거나 상담을 신청 |
| **주요 기능** | • 옵션명, 가격 노출<br>• VAT 포함/마취비 포함/사후관리비 포함 여부 뱃지 노출<br>• 부작용 안내 텍스트 노출<br>• 상세 이미지 노출<br>• 평점 및 리뷰 개수 진입 버튼<br>• **상담받기** 버튼(하단 고정) |

### 리뷰 종합 뷰 페이지

> **구현 기능:** `F004`, `F007` | **메뉴 위치:** 시술 상품 상세 페이지에서 진입

| 항목 | 내용 |
|------|------|
| **역할** | 상품에 대한 전체 리뷰 요약(관리자 입력)과 개별 리뷰 원문을 확인하는 화면 |
| **진입 경로** | 시술 상품 상세 페이지에서 리뷰 영역 클릭 |
| **사용자 행동** | 총 리뷰 수와 평균 평점, 관리자가 입력한 전체 긍정/부정 요약을 확인하고, 관심 있는 속성 태그를 선택하거나 개별 리뷰를 훑어봄 |
| **주요 기능** | • 총 리뷰 수, 평균 평점 노출<br>• 관리자가 직접 입력한 전체 긍정/부정 비율 도넛 차트 노출<br>• 관리자가 직접 입력한 전체 긍정/부정 요약 bullet 노출<br>• 속성 태그 5종(의료진/이용서비스/가격/시술효과/시술통증) 노출 및 선택 기능<br>• 개별 리뷰 카드 리스트 노출<br>• **구매하기**/**상담받기** 버튼(하단 고정) |
| **다음 이동** | 속성 태그 클릭 → 속성별 리뷰 필터 페이지, 하단 버튼 클릭 → 상담 신청 페이지 |

### 속성별 리뷰 필터 페이지

> **구현 기능:** `F005`, `F007` | **메뉴 위치:** 리뷰 종합 뷰 페이지에서 진입

| 항목 | 내용 |
|------|------|
| **역할** | 사용자가 선택한 속성(의료진/이용서비스/가격/시술효과/시술통증) 기준으로 관리자가 입력한 전용 요약과 관련 리뷰만 좁혀서 보여주는 화면 |
| **진입 경로** | 리뷰 종합 뷰 페이지에서 속성 태그 클릭 |
| **사용자 행동** | 선택한 속성에 대한 관리자 입력 요약(도넛 차트+bullet)을 확인하고, 해당 속성에 매핑된 개별 리뷰를 읽음. 다른 속성으로 전환하거나 전체 요약으로 복귀 가능 |
| **주요 기능** | • 선택된 속성명 표시<br>• 해당 속성 기준 관리자 입력 긍정/부정 비율 도넛 차트 노출<br>• 해당 속성 기준 관리자 입력 긍정/부정 요약 bullet 노출<br>• 해당 속성에 매핑된 개별 리뷰 카드만 필터링해 노출<br>• 다른 속성 태그로 전환하는 기능<br>• "전체 요약 보기" 버튼(리뷰 종합 뷰로 복귀) |
| **다음 이동** | "전체 요약 보기" 클릭 → 리뷰 종합 뷰 페이지, 다른 속성 태그 클릭 → 동일 페이지 내 속성 전환 |

### 상담 신청 페이지

> **구현 기능:** `F006` | **메뉴 위치:** 시술 상품 상세 페이지 또는 리뷰 종합 뷰 페이지에서 진입

| **주요 기능** | • 이름 입력<br>• 연락처 입력<br>• 희망일시 입력<br>• 신청 대상 상품 정보 표시<br>• **신청하기** 버튼 |
| **다음 이동** | 제출 성공 → 신청 완료 안내 후 시술 상품 상세 페이지로 복귀 |
